# UmaMusume Team Trials Tracker - Backend

Este repositorio contiene el backend de la aplicación UmaMusume Team Trials Tracker, construido con NestJS y Prisma.

## Despliegue

Recomendado para Railway, Render o VPS. Configura las variables de entorno necesarias.

## Instalación

```bash
npm install
npm run build
npm run start:prod
```

## Variables de entorno

- `DATABASE_URL`: URL de la base de datos PostgreSQL
- `JWT_SECRET`: Secreto para JWT
- `FRONTEND_URL`: URL del frontend (ejemplo: https://frontend-url)
